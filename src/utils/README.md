# 入选工具库的


# 命名约定

* 采用驼峰法命名,函数首字母一般是用小写, 类对象首字母大写
* 如果名称中含有常见的首字母缩写,则全部使用大写(如: ID 身份证号);
* 根据函数功能划分,推荐使用拟人、拟物化命名
* 动作类函数使用动词+名称名称

# 功能分类

* 数据判断验证类
* 数据解析转换类
* 数据查找、分析、排序类
* 数据加解密类 

# 关于类对象

对类对象进行函数化包装输出,避免使用时需要先使用new 操作符实例化,
保证工具库中的所有对象均为函数
