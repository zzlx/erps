# MS SQL Server

## Tedious

[Tedious](https://github.com/tediousjs/tedious) is a pure-Javascript 
implementation of the 
[TDS](https://github.com/tediousjs/tedious)(Tabular Data Stream Protocol)
which is used to interact with instances of Microsoft's SQL Server.

* TDS 7.4 (SQL Server 2012/2014/2016/2017)
* TDS 7.3.B (SQL Server 2008 R2)
* TDS 7.3.A (SQL Server 2008)
* TDS 7.2 (SQL Server 2005)
* TDS 7.1 (SQL Server 2000)
