[ERP Services(ERPs)](./docs/manual/README.md)
===============================

A ERP service system.

## 使用帮助

### 系统依赖安装

```
npm install
```

### 启动开发服务

编辑代码后自动重启服务器

```
npm run start:dev
```

### 启动ERP服务

```
npm run start
```

### 停止ERP服务

```
npm run stop
```

## 开发帮助

* 模块输出约定: index.mjs模块使用export方法输出, 不提供default模块
