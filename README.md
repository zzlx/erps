[ERPs](./docs/manual/README.md)
===============================

A ERP service system.

## 系统环境准备

* 使用nvm工具安装Node.js

```
nvm install 14.10.0
```

* 下载系统软件

```
git clone --depth 1 -b master https://github.com/zzlx/erps.git
```

* 安装系统依赖

```
npm install
```

## 目录简介

* bin
可执行脚本

* docs
系统文档

* public
前端资源及代码

* server
后端资源及代码

* src
源码库

## 开始使用

* 启动开发环境

```
npm run devel
```

* 启动服务

```
npm run start
```

* 停止服务

```
npm run stop
```
