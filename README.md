ERPS
====

ERPS(ERP Services Platform), 集成ERP、OA、CRM、HR等功能于一体的综合业务服务平台。

# 系统部署

请在类unix环境中安装部署

# 源码目录

* bin: 脚本程序
* docs: 文档
* public: 前端资源代码库
* resolvers: 解析器
* schema: schema
* services: 服务端程序
* src: 源码资源
* styles: 样式系统代码库
