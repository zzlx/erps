ERPs
====

一款企业ERP管理软件, 提供OA、CRM、会计、仓管以及档案管理等综合信息管理服务。

为企业管理提供一个基础软件平台，可以方便的集成各种应用,并提供标准化API数据接口服务。
此软件平台将围绕帮助企业管理资源，并有效利用资源进行开发。

# 开发目标

* Easy Deploy 易于部署
* Easy Use 易于使用
* Easy Maintainable 易于维护
* Easy Extend 易于扩展

# 使用帮助

> [Manual](./docs/READM.md)

* 启动EPRs

```
./bin/erps-daemon.mjs --start
```
